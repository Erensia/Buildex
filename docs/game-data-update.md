# 게임 데이터 갱신 기준

게임 데이터는 서버 계산의 단일 기준이다. 패치가 적용되면 캐릭터, 무기, 에코, 에코 세트, 주옵션을 같은 `data_version`과 `source_snapshot`으로 갱신한다.

1. 공식 패치 노트 또는 검증 가능한 게임 데이터 출처를 기록한다.
2. 신규 또는 변경된 데이터를 마이그레이션으로 추가하고, 모든 행에 출처 URL·스냅샷 날짜·데이터 버전을 넣는다.
3. `games.current_data_version`과 출처 정보를 같은 배포에서 바꾼다.
4. 계산·입력 검증·빌드 데이터 API가 변경값을 일관되게 읽는지 테스트한다.

플래너는 현재 사용 중인 데이터 버전과 검증 날짜, 출처 링크를 화면에 표시한다. 이로써 저장된 빌드의 `data_version`과 현재 게임 데이터를 비교할 수 있다.
